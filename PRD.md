# PRD - 产品需求文档

## 产品概述

**产品名称：** stibiums.top - 个人学术网站
**产品负责人：** 刘继轩 (Liu Jixuan)
**目标用户：** 导师、同行研究者、招聘方、技术社区

一个兼具个人展示和知识分享功能的全栈学术网站，基于 Next.js 构建，内置 CMS 管理后台，支持在线编辑发布内容。

---

## 核心功能

### 前台页面（公开访问）

| 功能 | 说明 |
|------|------|
| 首页 (About) | 个人简介、技能标签云、最近动态时间线、社交链接 |
| 博客 (Blog) | 文章列表、分类/标签筛选、分页、文章详情 |
| 学习笔记 (Notes) | 按课程分类组织、侧边栏导航、笔记详情 |
| 项目展示 (Projects) | 项目卡片网格、项目详情 |
| 学术出版物 (Publications) | 论文列表、BibTeX 引用复制 |
| 简历 (CV) | 结构化简历展示、PDF 下载 |
| 知识图谱 (Graph) | 笔记双向链接、D3.js 力导向图可视化 |
| 全文搜索 | Cmd+K 全局搜索弹窗，搜索博客和笔记 |
| 评论系统 | Giscus (基于 GitHub Discussions) |
| 点赞 | 匿名点赞（IP 防重复） |
| 访问统计 | 页面浏览量展示 |
| RSS 订阅 | /feed.xml |
| 中英双语 | 全站中英文切换 |
| 明暗主题 | 明/暗模式切换 |

### 管理后台（需认证）

| 功能 | 说明 |
|------|------|
| 统计仪表盘 | 访问量、点赞数、文章数等统计概览 |
| 文章管理 | 创建/编辑/删除/发布博客文章 |
| 笔记管理 | 创建/编辑/删除/发布学习笔记 + 分类管理 |
| 项目管理 | 创建/编辑/删除项目 |
| 出版物管理 | 创建/编辑/删除学术出版物 |
| 图片上传 | 编辑器内上传图片 |
| 站点设置 | 基础信息配置 |
| Monaco Editor | VS Code 内核的 Web Markdown 编辑器，支持语法高亮、迷你地图、自动保存 |
| Git 版本管理 | 每次保存自动 git add → commit → push，版本历史可追溯 |
| Markdown 上传 | 支持上传 .md 文件创建内容 |

## 用户故事

### 访客
- 作为访客，我可以浏览个人信息和作品集，了解站长背景
- 作为访客，我可以阅读博客和学习笔记，内容支持数学公式和代码高亮
- 作为访客，我可以通过 Cmd+K 搜索全站内容
- 作为访客，我可以通过知识图谱发现笔记之间的关联
- 作为访客，我可以对文章点赞和评论
- 作为访客，我可以通过 RSS 订阅内容更新
- 作为访客，我可以切换中英文界面
- 作为访客，我可以切换明暗主题
- 作为访客，我可以在手机上流畅浏览网站

### 管理员
- 作为管理员，我可以在网页后台直接创建、编辑、发布内容，无需修改代码
- 作为管理员，我可以在 Monaco Editor 中编辑 Markdown 文件，享受 VS Code 般的编辑体验
- 作为管理员，编辑保存后自动提交推送到 Git，版本历史可追溯
- 作为管理员，我可以管理文章的中英文双语内容
- 作为管理员，我可以查看访问统计和互动数据
- 作为管理员，我可以管理笔记的分类和双向链接

- Lighthouse 性能评分 > 90
- 所有页面 Mobile-first 适配

## 范围界定

### 包含
- 上述所有前台页面和管理后台功能
- Docker 部署配置
- Gitea Actions CI/CD
- GitHub 镜像仓库（用于 Giscus）

### 不包含（后续迭代）
- 邮件订阅服务（RSS 优先）
- 多用户管理（仅单管理员）
- 在线 CV 编辑（CV 内容硬编码或数据库存储）
- 邮件订阅服务（RSS 优先）
- 多用户管理（仅单管理员）
- 在线 CV 编辑（CV 内容硬编码或数据库存储）
